## Clone and Run the App

Clone the app:

```bash
git clone https://github.com/suyanksaraswat/react-typescript-hooks.git
cd react-typescript-hooks
```

Install dependencies:

```bash
yarn
```

Run the app:

```
yarn start
```

Run the tet:

```
yarn run test
```
